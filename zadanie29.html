1. W jaki sposób można określić właściwość danego obiektu?

Odpowiedź:

Sposób 1:

  • Używając kropki:
    auto.marka = "Opel Omega"; (tekst)
    auto.rok = 1996; (liczba)
    auto.cena = 25000; (liczba)

Sposób 2:

  • Używając nawiasów kwadratowych:
    auto["cena"] = 25000;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Odpowiedź:

Do pól i metod odwołujemy się jak do zwykłych zmiennych/funkcji, ale poprzedzamy nazwą obiektu i kropką:

console.log(samochod1.kolor);

3. Cechy charakterystyczne konstruktora

Odpowiedź:

  • Wywoływany automatycznie podczas tworzenia obiektu.
  • Nazywa się tak samo jak klasa.
  • Nie zwraca wartości.
  • Może przyjmować argumenty do ustawienia właściwości obiektu.

4. Konstruowanie obiektu (3 pola i 1 metoda)

Odpowiedź:

var osoba_1 = {
    nazwisko: 'Kowalski',
    imie: 'Jan',
    zawod: 'piekarz',
    wyswietl: function() {
        document.write(this.nazwisko + ' ' + this.imie);
    }
}

5. Znaczenie słowa this i oddzielanie pól i metod

Odpowiedź:

this – pozwala odwołać się do pola lub metody obiektu z wnętrza tego obiektu.
Pola i metody w obrębie obiektu oddzielamy przecinkami.

6. Konstruktor w JS (przykład)

Odpowiedź:

function klient(nazwisko_k, imie_k, zawod_k) {
    this.nazwisko = nazwisko_k;
    this.imie = imie_k;
    this.zawod = zawod_k;

    this.wypisz = function() {
        alert(this.nazwisko + ' ' + this.imie);
    }
}

7. Znaczenie słowa kluczowego new (przykład)

Odpowiedź:

Tworzy nowy obiekt na podstawie konstruktora:

var osoba1 = new klient('Kowalski', 'Jan', 'kierowca');
var osoba2 = new klient('Nowak', 'Anna', 'sekretarka');

8. Znaczenie słowa kluczowego prototype (przykład)

Odpowiedź:

prototype – pozwala dodać metody i właściwości do wszystkich obiektów utworzonych na podstawie konstruktora.

function klient() {
    this.nazwisko = 'Bielski';
    this.imie = 'Paweł';
}
klient.prototype.pisz_dane = function() {
    document.write(this.nazwisko + ' ' + this.imie);
}
klient.prototype.zawod = 'kierowca';

var osoba1 = new klient();
osoba1.pisz_dane(); // wypisze "Bielski Paweł"

Dzięki prototype każdy nowo utworzony obiekt klient będzie miał metodę pisz_dane i właściwość zawod.
